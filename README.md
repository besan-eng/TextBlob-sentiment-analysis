# TextBlob Sentiment Analysis

This project analyzes the sentiment of multiple sentences using Python and TextBlob.
It calculates polarity and determine text as Positive, Negative, or Neutral.

## How to Use
1. Install Python
2. Install python packages and Textblob
3. Run the script sentiment2.py
4. The program prints and calculate the sentiment and polarity for each sentence

## Requirements
 Python 3.14 (should also work on Python 3.13)
 TextBlob library
